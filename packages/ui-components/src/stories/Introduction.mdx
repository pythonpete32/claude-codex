# UI Components Library

Welcome to the **UI Components Library** for the Atomic Codex project!

This library provides a comprehensive set of React components for building chat interfaces and displaying tool results from Claude conversations.

## What's Included

### shadcn/ui Components
All the beautiful, accessible components from shadcn/ui are included and ready to use:
- **Button**, **Card**, **Input**, **Select**, and many more
- **Form** components with validation
- **Layout** components like **Tabs**, **Accordion**, **Sidebar**
- **Feedback** components like **Alert**, **Badge**, **Progress**

### Chat Item Components
Custom components designed specifically for displaying Claude tool interactions:
- **BashTool** - Display bash commands with syntax highlighting and status
- **Context7MCP** - Show Context7 MCP tool results
- **EditTool** - Display file editing operations
- And many more...

## Design System

### Colors
The components use Tailwind CSS with a consistent color palette supporting both light and dark themes.

### Typography
Clean, readable typography with proper hierarchy and spacing.

### Accessibility
All components follow accessibility best practices with proper ARIA labels, keyboard navigation, and screen reader support.
